1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. You should use constants if you know that you're never going to change the value. Variables should be used if you don't know wether the variable is constant, or if you know that you need to change it later. They cannot all be constants. A lot of them need to change later in the code. It's generally better (if you have to) to use only variables over only constants

    5. We can use an if-statement to check if the values are the same. 12.199999 is not the same as 12.2 in code (although it's practically the same value for us humans), so if you want to make it pass as "equal", you can create something called an "epsilon". You can then check the difference in value of the two numbers compared to the epsilon value.
